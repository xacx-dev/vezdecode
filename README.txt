1. ВАРИАНТ


Устанавливаем python 3.8, затем через встроенное приложение pip устанавливаем библиотеку requests.

Дальше переходим по ссылке - https://vkhost.github.io
и получаем токен для приложения "Kate mobile" (авторизуемся, и копируем значение токена из адресной строки).

Дальше открывает беседу в вк, с участниками ВЕЗДЕКОДА, и в адресной строке видим параметр sel=c###, где ### - цифры(длина может быть и больше 3 знаков), копируем числовое значение, без буквы "C"

Полученный токен и id чата вставляем в код программы - а именно в переменные token и chat_id.

Запускаем программу c названием finder_1.py и ожидаем завершения её работы.

В итоге мы получим id пользователя, у которого больше всего клика в этом чате, и id этих друзей.
